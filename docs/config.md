# 配置项

## 基础配置

### path

- 类型: `string`
- 默认值: `'/github'`

GitHub 服务的路径。

### appId

- 类型: `string`

### appSecret

- 类型: `string`

### redirect

- 类型: `string`

### messagePrefix

- 类型: `string`
- 默认值: `'[GitHub] '`

显示在每条消息前的文本。

## 高级配置

### promptTimeout

- 类型: `number`

### replyTimeout

- 类型: `number`

### requestTimeout

- 类型: `number`

### events

- 类型: `EventConfig`
